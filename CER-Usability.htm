Required

Mandatory fields to be denoted using a thick bar or * along with a "Required" announcement for screen Readers (Guideline 1.2.1 and 1.4.1).

Example

Wrap the label and input tags in a div with class span3. input should have classes form-control and required. form-control should be the first class, and required should be the last one.

<div class="span3">
    <label for="username" >Username:</label>
    <input id="username" name="user.username" type="text" class="form-control required" aria-required="true">
</div>
